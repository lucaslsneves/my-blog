# Inicializando

 Clone o repositório

`git clone git@github.com:lucasluz99/my-blog.git`

 Entre na pasta

`cd my-blog`

 Instale as dependências

 `yarn`
 
 Build

 `yarn build`

 Inicie a aplicação

`yarn develop`
